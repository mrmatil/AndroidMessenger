# AndroidMessenger

### Project for 2019 Android Class

It's simple Android Messenger Clone that is using Google Firebase as online realtime database with all messages. <br>
App uses RecyclerView to display messages downloaded from database.

## Built with: 

* Java 8
* Google Firebase API
